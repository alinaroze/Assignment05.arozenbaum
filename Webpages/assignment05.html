Assignment 05

Prototyping Integration Solutions Using Mashup Tools

Here you will discover how to identify, research, and keep a record of resources while developing and integration prototype.

Objectives:

  • Explore Mashup tools and simple data integration problems using Yahoo Pipes
  • Know how to register and setup developer access to public Web services.
  • Know how to setup a data file from a remote web server
  • Demonstrate a working understanding of APIs and the XML format of data files and RSS results.
  • Apply design principles for the pipes-and-filters integration design pattern and learn its limitations

Instructions:

  1. How to build an HTML webpage
  2. How to setup WinSCP to transfer files
  3. How to navigate Google Books
  4. How to create a Yahoo Pipe
  5. How to create a more complicated Yahoo Pipe
  6. How to retrieve a NY Times Bestsellers list API

Data References:

  • Google Bookshelf XML
  • Google Bookshelf
  • Favorites Bookshelf Pipe
  • Bookshelf vs NY Times Bestsellers Pipe
  • Bookshelf vs NY Times Bestsellers RSS Feed
