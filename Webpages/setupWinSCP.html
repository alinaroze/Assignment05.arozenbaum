WinSCP Set-Up and File Transfer

This gives you an interface to make the coding in HTML easier

First-of-all, you must download WInSCP.

WinSCP Set-Up

  1. Save the downloaded file to "Downloads."
  2. Double-click the .exe set-up file to run it.
  3. Accept all the defaults (recommended) unless you know what preferences you want
  4. Enter personal.utulsa.edu
    5. where it says: Host Name
  5. Now use your utulsa.edu login (i.e. alina-rozenbaum) and the password you use at the university
  6. Click login.

You may now drag and drop files from your computer to your server. And you may also create new HTML files and edit them more easily than in Command Prompt.
