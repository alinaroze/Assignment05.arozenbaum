Creating a Pipeline

Searching an XML Google Bookshelf using a Pipeline

These instructions will show and describe how to create a pipeline based on Feed Data from Favorites Bookshelf XML

This pipline will allow its user to search for any book on this bookshelf based on the author and title of the book.

You will need a Yahoo account and be logged into it in order to create this pipe. Instructions on how to create a Yahoo account can be found here .

Go to Yahoo Pipes . This is where you will be creating your pipeline.

STEP 1: Fetching the Pipeline Data

Here we are simply fetching the XML data fields from the Google Books XML.

  • Find Sources on the left-hand toolbar. Click on it.
  • Click, drag and drop Fetch Data onto the blue canvas.
  • In the URL field, type in the address of the XML to be searched. In this case it's:
    http://personal.utulsa.edu/~air196/xmlBookshelf.xml
  • In the Path to item list space enter: books.book
  • Find Operators on the left-hand toolbar. Click on it.
  • Click, drag and drop Filter onto the blue canvas.
  • Change Block to Permit
  • For the first rule fill in the first blank with item.contributor
  • Click the plus to the left of Rules to add a new rule
  • For the second rule fill in the first blank with item.title
  • Click on the little dot underneath Fetch Data and drag the link to the top of Filter to connect the two.

STEP 2: Making Pipeline Search Fields

Here were are creating search fields for the user so that they can look through our XML file for a book by Title or Author.

  • Find User inputs on the left-hand toolbar. Click on it.
  • Click, drag and drop Text Input twice onto the blue canvas.
  • Fill in the fields as follows for the first Text Input:
    • Name: Author.
    • Prompt: Author.
    • Position: 0.
  • Fill in the fields as follows for the second Text Input:
    • Name: Title.
    • Prompt: Title.
    • Position: 1.
  • Click on the little dot underneath Text Input: Author and drag the link to the right of the first rule in Filter to connect the two.
  • Click on the little dot underneath Text Input: Title and drag the link to the right of the second rule in Filter to connect the two.

STEP 3: Organizing the Search Results in Alphabetical Order by Author (optional)

Here is how to do exactly what it says: organize the results the user recieves by using our pipe, in alphabetical order.

  • Find Operators on the left-hand toolbar. Click on it.
  • Click, drag and drop Sort onto the blue canvas.
  • Fill in the first blank with item.contributor.
  • It should now read:item.contributor in ascending order.

Finally just make a link between Filter and Sort. And create a link between Sort and Pipe output.

STEP 4: Publishing your Pipe

Here is how to make your pipe visible for others to see.

  1. Click the Save button in the top right corner under the search bar
  2. Choose a name for your pipe (i.e. Favorites Bookshelf)
  3. Now click on Properties next to the Save button.
  4. Type in Description and whatever tags you would like the pipelien to have. Click OK.
  5. Click on Properties again, but this time click on the little blue link above OK that says Run Pipe....
  6. You are now at a new page. Above your pipe are four buttons. Click the one that says Publish.
  7. Click OK on the dialogue box. It is now published!
