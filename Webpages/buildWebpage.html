Building an HTML Website

CAUTION: These specific instructions apply to those that go to the University of Tulsa. For any other websites, they will have to be modified to fit your domain.

On this page, I will show you how to build an HTML website. For this, one will need an SSH. This is a cryptographic network protocol for secure data communication, remote command-line login, remote command execution, and other secure network services between two networked computers.

Once you have an SSH, retrieve puTTY.

puTTY Set-Up

  1. Save the .exe somewhere you can find it later.
  2. Now go to the file you saved puTTY in and start the application.
  3. Enter personal.utulsa.edu where it says: Host Name.
  4. Click Open.
  5. Now use your utulsa.edu login (i.e. alina-rozenbaum) and the password you use at the university
  6. Now that you are in the sever type in: ls.
  7. you should see a public_html directory. This is where your HTML webpage files will eb created.
  8. Type in cd public_html to go into the corresponding folder.
  9. In order to create a homepage for your website, type in: nano index.html.
  10. A new HTML page now opens. Here you can code the webpage in HTML. (For a directory in HTML go to W3Schools.)

NOTE: To create another HTML page for your site, simply type: nano pageName.html. And be sure to link them to the main or other page in your site for the user to find them!
